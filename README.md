# Console-Video-Rental-System

## Infomation
 기본적인 기능 구현

 정렬보기 메뉴를 추가. 각 메뉴의 전문을 볼 수 있다. 

 검색필터를 이용해 보다 원하는 결과를 보다 빠르게 볼 수 있음

 연체료가 부과되는 시스템과 포인트 시스템 구현

 파일 입출력을 사용해 보다 백업과 복원이 용이
